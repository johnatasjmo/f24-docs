## Overview

Workspace Settings control the configuration of your tenant — the name, currencies, and preferences that apply to everyone in your organization.

> Only users with an admin role in your workspace can change these settings.

---

## Accessing Workspace Settings

Click your workspace name in the top-left → **Settings**, or navigate to **Settings → Workspace** in the left navigation.

---

## Settings Available

### General

| Setting          | Description                                    |
| ---------------- | ---------------------------------------------- |
| Workspace name   | Your company's display name on the platform    |
| Country          | Your tenant's country of operation             |
| Default currency | Base currency for rate display (currently USD) |

### Currencies

Configure which currencies your workspace uses for quoting. You can enable multiple currencies and set a default.

> **Note:** Multi-currency support is available. Contact your Flete24 administrator to configure currencies beyond USD.

### Users

Manage who has access to your workspace. See team members, their roles, and invite or remove users.

Go to **Settings → Users** for user management.

### Profile

Update your personal display name and notification preferences. These are per-user, not per-workspace.

Go to **Settings → Profile** for personal settings.

---

## Common Issues

| Problem                   | Solution                                                           |
| ------------------------- | ------------------------------------------------------------------ |
| Can't edit workspace name | Only workspace admins can edit — contact your admin                |
| Currency not available    | Contact the Flete24 team to enable additional currencies           |
| User not receiving invite | Check that the email address is correct and ask them to check spam |
