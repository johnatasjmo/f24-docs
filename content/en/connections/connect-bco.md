## Overview

A connection links your tenant (as an agent/carrier) with a BCO (cargo owner) on the Flete24 platform. Once connected, you can share quotes directly with the BCO through the platform instead of sending files by email.

---

## Before You Start

- You need the BCO's email address to send them the invite code.
- The BCO does not need to have a Flete24 account yet — they can create one when redeeming the code.

---

## Steps

### Step 1 — Go to Connections

Click **Connections** in the left navigation.

### Step 2 — Click "New Invite"

Click **Generate Invite Code** in the top-right corner.

### Step 3 — Enter the recipient's email (optional)

If you enter an email address, Flete24 will send the invite code directly to the BCO. If you leave it blank, the code is shown on screen for you to copy and share manually.

### Step 4 — Copy or send the code

The invite code is a 6-character code (e.g. `X7KM2P`). It is valid for 7 days.

If you entered an email, the BCO receives a message from `noreply@flete24.com` with the code and instructions.

### Step 5 — BCO redeems the code

The BCO goes to **Connections → Enter Code** in their Flete24 account, enters the code, and the connection is established.

---

## What Happens Next

Once the connection is active, it appears in your Connections list with the BCO's tenant name and status. You can then publish quotes to them directly from the platform _(publishing coming soon)_.

---

## Managing Connections

From the Connections page you can:

- See all active and pending connections
- View the BCO's tenant name and connection date
- Revoke a connection if needed

---

## Common Issues

| Problem                     | Solution                                                         |
| --------------------------- | ---------------------------------------------------------------- |
| Code expired                | Generate a new invite code — they expire after 7 days            |
| BCO can't find "Enter Code" | They must be in their Flete24 account → Connections → Enter Code |
| Email not received          | Ask the BCO to check spam; resend by generating a new code       |
