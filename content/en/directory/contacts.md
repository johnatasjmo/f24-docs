## Contacts

Contacts are individual people — buyers, logistics managers, coordinators at client companies.

### Create a Contact

1. Go to **Directory → Contacts** → click **New Contact**
2. Fill in: first name, last name, email (required), phone, title (optional)
3. Click **Save**

### Link a Contact to a Company

1. Open the contact's detail page
2. Scroll to the **Companies** section → click **Add Company**
3. Search for the company by name and select it
4. The link is saved automatically

A contact can belong to multiple companies.

## Companies

Companies are the organizations you work with — shipping lines, other forwarders, BCO clients.

### Create a Company

1. Go to **Directory → Companies** → click **New Company**
2. Fill in: company name, country, company type (required), SCAC code (for shipping lines)
3. Click **Save**

### Company Types

| Type           | Description                                                 |
| -------------- | ----------------------------------------------------------- |
| VOCC           | Shipping line that operates vessels — e.g. MSC, Hapag-Lloyd |
| Agent          | Freight forwarder or logistics intermediary                 |
| BCO            | Cargo owner / importer / exporter                           |
| NVOCC          | Forwarder issuing their own bills of lading                 |
| Customs Broker | Customs clearance agent                                     |
| Other          | Anything else                                               |

> Company type matters: rate cards use VOCCs as carriers, BCOs appear in the connections flow.

## Common Issues

| Problem                                | Solution                                 |
| -------------------------------------- | ---------------------------------------- |
| Duplicate contact warning              | Search before creating — filter by email |
| Carrier not in rate card dropdown      | Create the company with type VOCC first  |
| Company not found in connection invite | Ensure type is BCO                       |
