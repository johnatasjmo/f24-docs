## What is the Directory?

The Directory is your tenant's address book. It stores the companies and contacts you work with — shipping lines, BCO clients, other forwarders, and port agents.

Data in the directory is used across the platform:

- **Rate cards** reference companies as carriers or agents
- **Quotes** reference companies as clients and contacts as recipients
- **Connections** link your tenant to a BCO company
- **AI extraction** matches document names against your directory to identify carriers and agents

---

## Directory vs Reference Data

|                | Directory                   | Reference Data                                       |
| -------------- | --------------------------- | ---------------------------------------------------- |
| Scope          | Your tenant only            | Platform-wide (shared)                               |
| Examples       | Your clients, your carriers | Global carrier list, charge aliases, container types |
| Who manages it | You                         | Flete24 platform admins                              |

Global carriers (VOCC list), charge codes (charge master), and carrier charge aliases are managed platform-wide and visible to all tenants. See [Reference Data](../reference-data/overview.md).
