## Overview

Flete24 is a freight visibility and management platform built for Latin America. It connects freight forwarders (agents/carriers) with cargo owners (BCOs) on a single platform — replacing spreadsheets, WhatsApp threads, and email chains with structured, trackable workflows.

---

## What It Does

| Capability                                                       | Who uses it          |
| ---------------------------------------------------------------- | -------------------- |
| Rate management — store, import, and organize freight rates      | Agents               |
| Quote builder — create multi-lane quotes from your rate library  | Agents               |
| AI rate extraction — extract rates from PDF/Excel/WhatsApp files | Agents               |
| Directory — manage contacts, companies, and relationships        | Agents               |
| Connections — link with BCO partners                             | Agents + BCOs        |
| Received quotes — view and compare quotes from agents            | BCOs _(coming soon)_ |
| Shipment tracking — milestone visibility per booking             | BCOs _(coming soon)_ |

---

## Two Sides of the Platform

**Agent side (available now)**
Freight forwarders and logistics service providers use Flete24 to manage rates, build quotes, and send them to clients.

**BCO side (coming soon)**
Cargo owners and importers/exporters use Flete24 to receive quotes, compare rates, and track active shipments.
